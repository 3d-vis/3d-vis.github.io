# 3D Data Viewer

Static Three.js viewers for loading and inspecting 3D data in the browser.

Pages:

- `index.html` - auto-detect drag-and-drop router plus viewer selector
- `ply.html` - `.ply` point clouds, meshes, and 3D Gaussian Splatting PLY files
- `glb.html` - `.glb` and embedded `.gltf` scenes

Run locally:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
